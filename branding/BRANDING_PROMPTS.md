## Visual Style Guide

**Brand Colors:**
- Primary: Dark Navy (#0a0e27)
- Accent: Bright Cyan (#00d4ff)
- Supporting: Deep Purple (#1a0d2e), Teal (#00fff5)
- Text: White (#ffffff), Light Gray (#e0e0e0)

**Typography:**
- Modern sans-serif (Roboto, Inter, SF Pro)
- Bold weights for titles
- Clean, readable

**Visual Elements:**
- Magnifying glass (scope/analysis symbol)
- Ascending bars (performance profile visualization)
- Bottleneck highlight (tallest bar in different color)
- Circuit board patterns (subtle, technical)
- Data flow visualizations
- Minimal gradients (subtle when used)

**Mood:**
- Professional - Enterprise-grade profiling tool
- Analytical - Data-driven, precise measurement
- Tech-forward - Modern Python development
- Accessible - Beautiful reports make profiling easy
- Part of unified ecosystem - Beacon HQ / Team Brain visual identity

## Color Psychology

**Why Dark Navy + Cyan?**
- Navy: Professional, trustworthy, stable
- Cyan: Technical, modern, precise
- High contrast: Excellent readability
- Tech aesthetic: Matches developer tools

**Symbolism:**
- Magnifying glass = Analysis, investigation, deep dive
- Ascending bars = Performance metrics, improvement
- Highlighted bar = Bottleneck identification
